# 1.2 — Transformation — Tier 1

> Part of Phase 1 — Excel System

## Purpose

Tier 1 transforms take the per-ticker outputs from the fetcher functions (Section 1.1) and combine them into one stacked table per financial statement type. Each combiner query loops all tickers from `tblIngest` on the `Control` sheet.

## How it works

The `Companies` query reads the ticker list from `tblIngest`. Each Tier 1 combiner:

1. Gets the ticker list from `Companies`
2. Invokes the corresponding `fn*` fetcher function for each ticker
3. Filters out any failed/empty results
4. Stacks all valid results into one table
5. Loads to the corresponding `ALL_*` sheet

| Combiner Query | Fetcher Called | Output Sheet |
|---|---|---|
| `ALL_IS` | `fnIS` | ALL_IS |
| `ALL_BS` | `fnBS` | ALL_BS |
| `ALL_CFS` | `fnCFS` | ALL_CFS |
| `ALL_Ratio` | `fnRatio` | ALL_Ratio |
| `ALL_Beta` | `fnBeta` | ALL_Beta |
| `ALL_ForwardEst` | `fnForwardEst` | ALL_ForwardEst |

The `FRED` query operates independently — it reads series IDs from `tblFREDSeries` on the `Control` sheet and calls `fnFRED` per series. Output loads to `ALL_Rates`.
