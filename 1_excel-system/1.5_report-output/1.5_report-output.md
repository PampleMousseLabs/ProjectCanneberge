# 1.5 — Report Output

> Part of Phase 1 — Excel System | Status: Complete/always in progress

## Purpose

The report output layer presents the valuation results in a user-facing dashboard with projection controls, comparable analysis, and a consolidated valuation summary.

## Primary output surface

### Dash_Prjctn

The main model interface. Combines input controls and output visualization on a single sheet:

- **Projection controls** — toggle projection assumptions (via `modProjectionToggle`)
- **WACC inputs** — inline WACC parameter controls
- **Market Approach section** — GPC and GT multiple selection and weighting
- **Comparable stats dashboard** — GPC and subject company toggle tables driven by `RatioCatalog`
- **Dynamic comparison chart** — refreshed via `modCharts` / `Refresh Comp Chart` ribbon button
- **Reconciliation of values** — visual comparison of valuation approaches (DCF, GPC, GT, NAV)

### Summary

Consolidated valuation summary. Currently a candidate for deletion — its functionality may be absorbed into `Dash_Prjctn`.

## Supporting sheets

| Sheet | Role |
|---|---|
| `Chart Helper` | Structures data for chart consumption |
| `RatioCatalog` | Ratio definitions feeding the toggle tables on `Dash_Prjctn` |

## Key files

- Sheets: `Dash_Prjctn`, `Summary`, `Chart Helper`, `RatioCatalog`
- VBA: `modCharts.bas`, `modProjectionToggle.bas`
